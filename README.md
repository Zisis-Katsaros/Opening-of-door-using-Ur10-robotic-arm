# Opening of door using Ur10 robotic arm

As part of my Robotics course I was tasked in designing the unlocking and opening of a door using the Ur10 robotic arm, which has 6 degrees of freedom. You can read the documentation (report) for this project [here](docs.md). Although it is in Greek, there are a handful of calculations and figures, therefore non-Greek-speaking readers can still get a sence of the project.
